# PosingDemo

Гид по позированию для фотосессий. Каталог поз с фильтрацией, избранным, случайными позами и советами.

## Tech Stack

- **Next.js 16** — App Router
- **Tailwind CSS v4** — стилизация
- **shadcn/ui** — UI-компоненты
- **TypeScript** — типизация

## Запуск

```bash
npm install
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000).

## Страницы

| Путь | Описание |
|------|----------|
| `/` | Главная с featured-позами и категориями |
| `/poses` | Каталог с фильтрацией (категория, сложность, пол, поиск) |
| `/poses/[id]` | Детальная страница позы |
| `/random` | Случайная поза |
| `/favorites` | Избранное (localStorage) |
| `/tips` | Советы по позированию |
